Reference desk

Working glossary

A compact vocabulary for reading the atlas and discussing visualization decisions with precision.

Aesthetic mapping
A rule connecting a data field to a visual property such as position, color, size, or shape.
Baseline
The reference point from which a quantity or change is visually measured.
Calibration
Agreement between predicted probabilities and observed frequencies.
Channel
A visual property used to encode information.
Conditioning
Examining a relationship separately within levels or ranges of another variable.
Data domain
The set or range of values represented by a scale.
Faceting
Splitting data into repeated panels using categorical variables.
Geom / mark
The visible object used to represent data, such as a point, line, or rectangle.
Glyph
A compound symbol whose visual features encode multiple variables.
Interquartile range
The distance between the first and third quartiles.
Leverage
The degree to which an observation has unusual predictor values and can influence a fitted model.
Overplotting
Loss of information when marks overlap.
Preattentive feature
A visual difference detected rapidly before focused inspection.
Residual
Observed value minus model-predicted value.
Scale
The function mapping data values into visual values.
Small multiple
A repeated chart design shown across subsets for comparison.
Sparkline
A compact, word-sized time-series graphic.
Uncertainty interval
A range expressing plausible values under stated assumptions.
Visual hierarchy
The designed order in which elements attract attention.
Working memory
The limited mental workspace used to hold and compare active information.
Further study

Core reading map

This atlas synthesizes major ideas from foundational and modern work. The entries below are intentionally concise so the website remains a teaching guide rather than a substitute for the original works.

Cleveland & McGill

Graphical perception, elementary perceptual tasks, and evidence-based ranking of visual encodings.

John Tukey

Exploratory data analysis, resistant summaries, transformations, re-expression, and learning from residual structure.

William Cleveland

Statistical graphics, scales, labeling, smoothing, residuals, conditioning, and multivariate reasoning.

Edward Tufte

Graphical integrity, data-ink, chartjunk, small multiples, dense evidence, and visual explanation.

Stephen Few

Dashboard design, preattentive attributes, operational monitoring, and practical business intelligence.

Wilkinson & Wickham

The grammar of graphics and layered systems that separate data, mappings, marks, statistics, scales, coordinates, and facets.

Colin Ware

Human vision, color, attention, visual working memory, and the cognitive foundations of information visualization.

Heer, Bostock & collaborators

Large-scale graphical perception studies, interactive visualization, animation, crowdsourcing, and modern empirical validation.

Modern model explanation

SHAP, partial dependence, ICE, calibration, dimensionality reduction, decision curves, and uncertainty communication.
